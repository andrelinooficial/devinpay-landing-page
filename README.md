# DevinPay - Landing Page

Uma landing page moderna, responsiva e profissional desenvolvida para a apresentação do produto **DevinPay**. Focada em conversão, usabilidade e em proporcionar uma experiência de usuário fluida.

## 🚀 Funcionalidades

- **Design Responsivo:** Layout adaptável para todas as telas (desktops, tablets e smartphones).
- **Animações de Scroll:** Elementos surgem suavemente na tela à medida que o usuário rola a página, proporcionando dinamismo (implementado via `Intersection Observer`).
- **Iconografia Elegante:** Utilização da biblioteca Lucide Icons para ícones consistentes e de alta qualidade.
- **Alta Performance:** Código limpo, em Vanilla JS, e sem dependências pesadas para garantir um carregamento ultrarrápido.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Estilização, animações (`scale-in`, `is-visible`) e media queries para responsividade.
- **JavaScript (Vanilla):** Lógica para animações de interface no scroll e inicialização da renderização dos ícones.

## ⚙️ Como executar o projeto

Por ser um projeto frontend estático, não é necessária a instalação de dependências ou ambientes de execução complexos.

1. Clone este repositório na sua máquina local:
   `git clone https://github.com/seu-usuario/devinpay-landing-page.git`
2. Acesse a pasta do projeto:
   `cd devinpay-landing-page`
3. Abra o arquivo `index.html` no seu navegador web de preferência.
   
> **Dica para desenvolvedores:** Para visualizar edições em tempo real, recomenda-se abrir a pasta do projeto no VS Code e utilizar a extensão **Live Server**.
